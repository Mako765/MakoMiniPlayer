# MakoMiniPlayer

**Lightweight Android TV & Android Box video player with full subtitle support.**

## Why MakoMiniPlayer?

Most Android TV boxes struggle with heavy, resource-demanding video players. 
MakoMiniPlayer is built to be fast and lightweight — designed specifically for 
Android TV and Android Box devices that slow down with heavier apps.

Works as both an **external player** (launched from IPTV apps like TiviMate) 
and as a **standalone player** for local and network files.

## Features

- ✅ Plays HLS, HTTP, local and network streams
- ✅ Auto-loads subtitles from the same folder as the video
- ✅ Subtitle search via **OpenSubtitles.com** and **SubDL** (free registration)
- ✅ Manual subtitle loading from local storage or network
- ✅ Full subtitle customization:
  - Font size, color, bold
  - Shadow size and outline width
  - Dark background
  - Position adjustment (black bar support)
  - Text encoding (UTF-8, Windows-1250, and more)
- ✅ 19 aspect ratio / resize modes
- ✅ Playback speed control
- ✅ Works as external player for IPTV apps
- ✅ Google TV and Android Box launcher icon support
